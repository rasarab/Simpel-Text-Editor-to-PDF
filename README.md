# Simpel-Text-Editor-to-PDF
Html text editor dengan konversi ke pdf

Kamu bisa pakai file HTML tunggal ini—ada editor sederhana (contenteditable), opsi judul/filename, ukuran kertas, margin, dan tombol “Export PDF”. Tidak perlu backend. Cukup buka di browser modern.

Fitur ringkas
- Editor rich text: bold, italic, underline, heading, list, align, link.
- Metadata PDF: judul dokumen, nama file, ukuran kertas, margin.
- Konversi PDF lokal: menggunakan jsPDF + html2canvas (CDN).
